# Student Voting System

A web-based student voting system built using Java, JSP, Servlets, HTML, CSS, and MySQL.

## Features
- Student Login & Signup
- Admin Login
- Add Candidates
- Add Positions
- Secure Voting System
- Results Display

## Tech Stack
- Java
- JSP & Servlets
- HTML/CSS
- MySQL
- Maven
